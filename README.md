# Snake Game

A classic Snake game built with HTML, CSS, and JavaScript. This project demonstrates fundamental web development concepts and game programming techniques.

## Features

- Responsive design that works on desktop and mobile devices
- Score tracking with local storage for high score persistence
- Gradually increasing difficulty as your snake grows longer
- Clean, modern UI with intuitive controls
- Visual grid system for better gameplay visibility

## How to Play

1. Open `index.html` in any modern web browser
2. Click the "Start Game" button to begin
3. Use the arrow keys to control the snake's direction
4. Eat the red food to grow longer and earn points
5. Avoid hitting the walls or your own tail
6. Try to beat your high score!

## Game Controls

- **Arrow Up**: Move snake up
- **Arrow Down**: Move snake down
- **Arrow Left**: Move snake left
- **Arrow Right**: Move snake right
- **Start Game Button**: Start or restart the game
- **Reset Game Button**: Reset the game to its initial state

## Technical Details

This game is built using:

- **HTML5 Canvas** for rendering the game
- **CSS3** for styling and responsive design
- **JavaScript** for game logic and interactivity
- **Local Storage API** for saving high scores

## Project Structure

- `index.html` - Main HTML file with game structure
- `style.css` - CSS styles for the game interface
- `script.js` - JavaScript code containing all game logic
